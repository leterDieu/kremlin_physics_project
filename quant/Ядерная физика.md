#subsection
> Ядерная физика

> [!note]+ Обозначения и термины
> Нуклон - протон или нейтрон.
> 
> **Описание ядра**
> $^A_ZX$ - форма записи какой-то частицы $X$, где
> $A$ - массовое число (число нуклонов)
> $Z$ - заряд ядра
> 
> $N$ - число нейтронов
> $N = A - Z$
> 
> $p$ - протон ($^1_1p$)
> $n$ - нейтрон ($^1_0n$)
> $e$ - электрон ($^{\space\space\space0}_{-1}e$)
> 
> Изотопы - ядра одного элемента, имеющие разное количество нейтронов

> **Ядерные реакции**
>> [!note]+ Распады
>> $\alpha$-распад - вылет из ядра $^4_2He$
>> $\beta$-распад - вылет из ядра $^{\space\space\space0}_{-1}e$
>> $\gamma$-распад - вылет из ядра энергии в виде $h\nu$
> 
>> [!note]+ Пример записи реакций
>>  $^{A_1}_{Z_1}X$ -> $^{A_2}_{Z_2}Y + d\space^{\space\space\space0}_{-1}e$
>>  Ядро $X$ пережило $d$ $\beta$-распадов. Соответственно:
>>  $A_1 = A_2$
>>  $Z_1 = Z_2 - d$
>>  ---
>> $^{A_1}_{Z_1}X$ -> $^{A_2}_{Z_2}Y + d\space^4_2He$
>>  Ядро $X$ пережило $d$ $\alpha$-распадов. Соответственно:
>>  $A_1 = A_2 + 4d$
>>  $Z_1 = Z_2 + 2d$
>>  ---
>> Аналогично с нейтронами (облучение нейтронами и вылет нейтронов) и распадом одного ядра на два или несколько ядер разных элементов

> **Полураспад**
> $T$ - период полураспада - время, за которое распадается половина ядер
> $N$ - число нераспавшихся ядер
> $N_0$ - изначальное число ядер
> 
> $N = N_0 * 2^{-\frac{t}{T}}$
