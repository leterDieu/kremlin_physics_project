>[!example] Основы
>
>> [!abstract]  Скорость света
>> $c = const \approx 300.000 \frac{\text{м}}{\text{с}}$ (вакуум); 
>>
>> скорость света в среде меньше, чем в вакууме (всегда $n \geq 1$)
>> $n = \frac{c}{\mathcal{v}}$
>>> где:
>>> $\mathcal{v}$ - скорость света в среде
>>> $n$ - абсолютный показатель преломления вещества (среды)
>
>> [!abstract]  Свет - электромагнитная волна
>> Как для волны, для неё истинно:
>> $\lambda = \frac{c}{\nu}$
>>> где:
>>> $\lambda$ - длина волны
>>> $\nu$ - частота волны

>[!example]+ Закон параксиальной оптики
>
> Для малых (параксиальных) углов верно соотношение:
> $\sin{\alpha} = \tan{\alpha} = \alpha$
>
> *note: Глаз человека воспринимает только параксиальные лучи*

>[!example]+ Законы отражения света
> Угол падения луча - величина угла между перпендикуляром к плоскости и падающим лучом
>
>> [!abstract]  Первый закон
>> Падающий и отражённый лучи лежат в одной плоскости с перпендикуляром к отражающей поверхности (т.е. они фотоны после отражения не летят куда попало)
>
>> [!abstract]  Второй закон
>> Угол падения равен углу отражения

>[!example]+ Законы преломления света
>
>> [!abstract]  Первый закон
>> Падающий и преломлённый лучи лежат в одной плоскости с перпендикуляром к преломляющей поверхности (то же самое что и с отражением)
>
>> [!abstract]  Второй закон
>> $n_1 \cdot sin(\alpha) = n_2 \cdot sin(\beta)$
>> где $n_1$ - абс.показ.прел. в первой среде; $\alpha$ - угол падения в первой среде
>
>> [!abstract]  Относительный показатель преломления
>> $n_\text{отн} = \frac{n_2}{n_1} = n_\text{21}$

>[!example]+ Полное внутреннее отражение
> явление, когда свет из переходит из среды 1 в среду 2 с условием $n_1 > n_2$ и при этом угол падения больше критического
> фактически, предельный это тот угол, когда свет преломляется под углом $90^\circ$
>
>> [!abstract]  Предельный угол
>> $sin(\alpha) = n_\text{отн}$
>> т.е. $sin(\beta) = 1$


>[!example]+ Линзы
>
>> [!abstract] Фокус
>> Фокус ($F$ - фокусное расстояние)- точка, где собирается параллельный световой пучок, падающий перпендикулярно линзе (т.е. место куда все перпендикулярные лучи преломляются); для любой линзы их два: передний и задний
>> Фокальная плоскость - плоскость параллельная линзе и отступающая от неё на расстояние F. (их тоже две для линзы)
>>
>> Оптическая сила линзы: $D = \frac{1}{D}$
>
>> [!abstract] Собирающая и рассеивающая
>> фактически это система из двух призм. Как известно, призма **всегда** отклоняет свет к основанию; потому (В ЧАСТНОМ СЛУЧАЕ!):
>> - собирающая линза состоит из двух призм с общим основанием
>> - рассеивающая линза состоит из двух призм с общей вершиной 
>> *note: это скорее для понимания, на экзамене такое не нужно*
>>
>> Из этого также следует, что луч, прошедший через середину линзы (то есть пересечение обеих призм), не преломляется
>
>> [!abstract] Оптические оси
>> 1. Главная: прямая-перпендикуляр к плоскости линзы, опущенный в её середину. Луч, лежащий на этой оси не преломляется
>> 2. Побочная: прямая, проходящая через середину под любым углом.
>
>> [!abstract] Действие линзы
>> линза отклоняет световой пучок так, что луч, попавший на линзу, должен пересечь фокальную плоскость там же, где пересечёт её побочная оптическая ось параллельная лучу
>>
>> в случае рассеивающей линзы, должны пересечься продолжение (назад) преломлённого наружу луча и побоч. оптич. ось
>
>> [!abstract] Пример преломления луча собирающей линзой
>> ![[lense.png]]
>
>> [!abstract] Формула тонкой линзы:
>> **Важно:** в начале задач на оптику стоит сказать, что так как линза тонкая - формула выполняется.
>>
>> $\frac{1}{F} = \frac{1}{d} + \frac{1}{f}$
>> где:
>> $f$ - длина перпендикуляра от источника до линзы (= расстояние до источника)
>> $d$ - длина перпендикуляра от изображения до линзы (= расстояние до изображения)
>>> [!warning]+ **Важно!**
>>> В стандартной оптике эти величины могут быть положительными и отрицательными - для мнимых фокусов, изображений и источников. Однако для упрощения собственных вычислений (в школьной программе в рф в основном так и делают), рекомендую считать величины положительными, а знаки ставить в формуле по характерным особенностям (далее о них). В ЕГЭ такое допустимо, пояснений **не требует**. Например для рассеивающей линзы с мнимым изображением: $-\frac{1}{F} = \frac{1}{d} - \frac{1}{f}$
>>> $\pm\frac{1}{F}$ - плюс, если линза собирающая. минус, если рассеивающая 
>>> $\pm\frac{1}{d}$ - плюс, если источник действительный (является точкой испускающей лучи). минус, если мнимый - является пучком сходящихся лучей (то есть когда они падают на линзу эти лучи ещё не сошлись) 
>>> $\pm\frac{1}{f}$ - плюс, если изображение действительное. минус, если мнимое.
>>> 
>>> определить, каким конкретно будет изображение (мнимым или действительным) можно из соображений расположения источника - ближе фокусного расстояния; на фокусном расстоянии; между F и 2F и так далее - какое изображение соответствует каждому из перечисленных условий можно найти в инете. 
>>> ***Важно:*** зная какая линза и какой источник, на черновике по формуле тонкой линзы можно понять какое будет изображение: если перед 1/f математически получился минус - оно мнимое, в формуле на чистовике можно указать минус (это следует из условия: в нём скорее всего дано, где относительно фокуса располагается источник)
>
>> [!abstract] Увеличение
>> $\Gamma = \frac{H}{h}$
>> где $H$ и $h$ - высота изображения и источника соответственно.
>> Также важный факт (не требует дополнительных пояснений), следующий из подобия:
>> $\Gamma = \frac{f}{d} = \frac{H}{h}$
>
>> [!abstract] полезные выводы
>> если точка удалена от линзы на $2F$, то её изображение также удалено от неё на $2F$. - это требует простого доказательства через формулу тонкой линзы

>[!example]+ Физическая оптика
> когерентный источник - источник электро-магнитных волн, колеблющихся с одной частотой и сохраняющих постоянную фазу
> интерференция света - из двух когерентных источников на удалении друг от друга до равноудалённой от них плоскости световая эм-волна доходит с разными фазами колебания; из-за чего амплитуда волн увеличивается (если фазы одинаковы у сошедшихся волн) или уменьшается (если фазы противоположны) в зависимости от расположения точки наблюдения.
> 
> $\Delta r$ - разность хода. $\Delta r = | r1 - r2 |$; разность расстояний до точки от источников
>
>> [!abstract] Интерференционные максимумы и минимумы
>> условие максимума в точке: $\Delta r = 2n \cdot \frac{\lambda}{2}$ - т.е. в точку приходит чётное количество ПОЛУ-волн
>> условие минимума в точке: $\Delta r = (2n+1) \cdot \frac{\lambda}{2}$ - т.е. в точку приходит нечётное количество ПОЛУ-волн
>
>> [!abstract] Дифракция света
>> ***честно, сам не до конца понимаю эту тему. могу сильно ошибаться где-то***
>> - световая волна способна заходить в область геометрической тени.
>>
>> Дифракционная решётка - "щели" с определённым и постоянным периодом (как расчёска)
>> $d$ - период решётки. Расстояние между соседними щелями
>> $\varphi$ - угол между перпендикуляром к решётке и направлением на максимум
>> $k$ - порядок максимума. Первый (центральный) имеет значение 0
>> $\lambda$ - длина волны
>> $d \cdot sin(\varphi) = k\lambda$
>>
>> Из этого также следует, что существует ограниченно количество максимумов ($sin \leq 1$). также, можно найти последний период максимума.
>
>> [!abstract] Дисперсия света
>> белый свет - не монохроматический; он состоит из разных с разной частотой.
>> 
>> Дисперсия - явление, при котором видно, что **показатель преломления среды разный для разных частот световой волны**
>> *красные лучи отклоняются меньше, чем фиолетовые*
