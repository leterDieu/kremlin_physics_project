>[!example]+ основные определения
>
>> [!abstract]  количество вещества
>>> $ν$ - Количество вещества. СИ[$ν$] = моль
>> $M$ - Молярная масса. СИ[$M$] = кг/моль
>> $N_A$ - Число Авагадро СИ[$N_A$] = 1/моль
>> 
>> $ν = \frac{m}{M} = \frac{N}{N_A}$ 
>
>> [!abstract]  концентрация
> $V$ - Объем. CИ[$V$] = м^3
> $n$ - Концентрация. СИ[$n$] = 1/м^3
> $n = \frac{N}{V}$
>
>> [!abstract] Масса одной молекулы
> $m_0$ - Масса одной молекулы. СИ[$m_0$] = кг
> $m = Nm_0$
>
>
> $ρ$ - Плотность. СИ[$ρ$] = кг/м^3
> $ρ = \frac{m}{V}$
>
> $T$ - Абсолютная температура. СИ[$T$] = К
> $T = t°С + 273K$
>
> $k$ - Постоянная Больцмана. СИ[$k$] = Дж/К
>
> $R$ - Молярная газовая постоянная. СИ[$R$] = Дж/(моль * К)
> $R = kN_A$
>
> $p$ - Давление. СИ[$T$] = Па
> 
> $\bar{\mathcal{V}}$ - Средняя скорость молекул. СИ[$\bar{\mathcal{V}}$] = м/c
> $\bar{E}_k$ - Средняя кинетическая энергия одной молекулы. СИ[$\bar{E}_k$] = Дж
>
>> [!abstract] Средняя кинетическая энергия; давление
> $\bar{E}_k = \frac{m_0\bar{\mathcal{V}}^2}{2} = \frac{3}{2}kT$ 
> $p = \frac{1}{3}nm_0\bar{\mathcal{V}}^2 = \frac{2}{3}n\bar{E}_k$

>[!example]+ основные формулы
>
>> [!abstract] **Основное уровнение МКТ**
>> $p = nkT$
>
>> [!abstract] **Уравнение Менделеева-Клапейрона**
> $pV = νRT = \frac{mRT}{M}$
> $p = \frac{ρRT}{M}$

> [!example]+ Вывод формул для средней скорости молекул
> $\bar{\mathcal{V}} = \sqrt{\frac{3kT}{m_0}} = \sqrt{\frac{3RT}{M}}$ - данные формулы нельзя использовать без выводов:
> 
> $\bar{E}_k = \frac{m_0\bar{\mathcal{V}}^2}{2} = \frac{3}{2}kT$  => $\bar{\mathcal{V}} = \sqrt{\frac{3kT}{m_0}}$
> $m = Nm_0$, $\bar{\mathcal{V}} = \sqrt{\frac{3kT}{m_0}}$ => $\bar{\mathcal{V}} = \sqrt{\frac{3kNT}{m}}$ ; $ν = \frac{m}{M} = \frac{N}{N_A}$, $R = kN_A$  => $\bar{\mathcal{V}} = \sqrt{\frac{3kNT}{νM}} = \sqrt{\frac{3kN_AT}{M}} = \sqrt{\frac{3RT}{M}}$ ;

> [!example]+ **Изопроцессы, газовые законы**  
>> [!abstract] Изотермический
>>$T=const$.  
> По закону Бойля-Мариотта $p \sim \frac{1}{V}$
>
>> [!abstract] Изохорный
> $V=const$.  
> По закону Шарля $p \sim T$
>
>> [!abstract] Изобарный
> $p=const$.  
> По закону Гей-Люссака $V \sim T$
>
>> [!note]+ Запоминалка
>>Бойля-Мариотта - боил - кипятить (изотермический)  
>>Шарля - шар - объем (изохорный)

>> [!warning]+ На практике
>> 1. Только из того, что один из макропараметров ($p$, $V$ или $T$) константа **и $ν$ константа** следует, что данный процесс - изопроцесс. (То есть следует писать $p=const$, $ν=const$ => 1-2 - изобарный процесс). 
>> 2. Обосновывать зависимость одного макропараметра от другого в изопроцессе можно как и уровнением Менделеева-Клопейрона, так и соответствующим газовым законом
>> 3. Часто в задаче прямая на графике (например $p$-$T$) может проходить через начало координат. Это означает, что величины прямопропорциональны (например $p \sim T$), следовательно это изопроцесс. В решении эти наблюдения нужно прописывать следующим образом: "Из графика видно, что прямая 1-2 проходим через начало координат => $p = aT$, где $a=const$. По закону Менделеева-Клопейрона $pV=νRT$ => $aTV=νRT$ => $V = \frac{νR}{a}$, где $ν=const$, $R=const$, $a=const$ => $V=const$ => процесс изохорный"
>> 4. Иногда в задачах попадаются нестандартные координаты, например $n$ или $ρ$. В таких случаях применимо все, что написано в пункте 3, с учетом того, что $n = \frac{N}{V}$ и $ρ = \frac{m}{V}$, соответственно $n \sim \frac{1}{V}$ и $ρ \sim \frac{1}{V}$ при $N=const$ и $m=const$
>> 5. На графике в координатах $p$-$V$ изотерма - это гипербола (т.к. $p \sim \frac{1}{V}$)
>> 6. При решении задач с графиками, если дан масштаб, следует говорить насколько увеличивается/уменьшается тот или иной параметр
>> 7. В задачах с поршнем с газом обычно происходят только изобарные процессы. Это следует из того, что на поршень вниз действуют неизменные силы атмосферного давления и силы тяжести. Так как в любой момент времени поршень находится в равновесии, сумма этих сил равна силе давления газа под поршнем, а значит она константна
