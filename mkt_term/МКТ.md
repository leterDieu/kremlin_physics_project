#subsection

> МКТ

> [!note]- Обоснования для №21
> В решении задачи следует написать, что формулы применяются к модели идеального одноатомного газа.

> $ν$ - Количество вещества. СИ[$ν$] = моль
> $M$ - Молярная масса. СИ[$M$] = кг/моль
> $N_A$ - Число Авагадро СИ[$N_A$] = 1/моль
> $ν = \frac{m}{M} = \frac{N}{N_A}$ 

> $V$ - Объем. CИ[$V$] = м^3
> $n$ - Концентрация. СИ[$n$] = 1/м^3
> $n = \frac{N}{V}$

>>[!note] Литры <-> Кубометры
>>1 л = 1 дм$^3$ = $10^{-3}$ м$^3$
>>1 м$^3$ = 1000 л

> $m_0$ - Масса одной молекулы. СИ[$m_0$] = кг
> $m = Nm_0$

> $ρ$ - Плотность. СИ[$ρ$] = кг/м^3
> $ρ = \frac{m}{V}$

> $T$ - Абсолютная температура. СИ[$T$] = К
> $T = t°С + 273K$

> $k$ - Постоянная Больцмана. СИ[$k$] = Дж/К

> $p$ - Давление. СИ[$T$] = Па
> $\bar{\mathcal{V}}$ - Средняя скорость молекул. СИ[$\bar{\mathcal{V}}$] = м/c
> $\bar{E}_k$ - Средняя кинетическая энергия одной молекулы. СИ[$\bar{E}_k$] = Дж
> $\bar{E}_k = \frac{m_0\bar{\mathcal{V}}^2}{2} = \frac{3}{2}kT$ 
> $p = \frac{1}{3}nm_0\bar{\mathcal{V}}^2 = \frac{2}{3}n\bar{E}_k$

> $p = nkT$ - **Основное уровнение МКТ**

> $R$ - Молярная газовая постоянная. СИ[$R$] = Дж/(моль * К)
> $R = kN_A$

> **Уравнение Менделеева-Клапейрона**
> $pV = νRT = \frac{mRT}{M}$
> $p = \frac{ρRT}{M}$

> [!note]+ Вывод формул для средней скорости молекул
> $\bar{\mathcal{V}} = \sqrt{\frac{3kT}{m_0}} = \sqrt{\frac{3RT}{M}}$ - данные формулы нельзя использовать без выводов:
> 
> $\bar{E}_k = \frac{m_0\bar{\mathcal{V}}^2}{2} = \frac{3}{2}kT$  => $\bar{\mathcal{V}} = \sqrt{\frac{3kT}{m_0}}$
> $m = Nm_0$, $\bar{\mathcal{V}} = \sqrt{\frac{3kT}{m_0}}$ => $\bar{\mathcal{V}} = \sqrt{\frac{3kNT}{m}}$ ; $ν = \frac{m}{M} = \frac{N}{N_A}$, $R = kN_A$  => $\bar{\mathcal{V}} = \sqrt{\frac{3kNT}{νM}} = \sqrt{\frac{3kN_AT}{M}} = \sqrt{\frac{3RT}{M}}$ ;
