>[!example] Основы
> Материя - объективная реальность (то, что окружает нас и находится вне зависимости от нашего сознания); занимает пространство и при этом имеет разные характеристики
>
> Физическое поле - особый вид материи, не имеющий массы покоя и не состоящий из привычных прочей материи атомов, но существующий в каждой точке объективной действительности и осуществляющий взаимодействие между телами на расстоянии. Иными словами, поле - передатчик взаимодействий.
> ***как и математическое, является множеством единиц (пространства), для которых определены некоторые математические операции***

>[!example] Заряд
>
> Фундаментальное свойство материи, определяющие возможность и величину участия в электро-магнитных взаимодействиях; скалярная величина.
> 
> Электризация - процесс получения заряженных тел из нейтральных, при котором происходит перераспределение электрических зарядов (основным их носителем является электрон)
>
>> [!abstract]  **Закон сохранения электрического заряда**
>> Заряд не может появится из ниоткуда и уйти в никуда
>> $q_\text{сист нач} = q_\text{сист конеч}$

>[!example] Закон Кулона
> Любые два заряженных тела взаимодействуют друг с другом
>
>> [!abstract]  формула
>> $F = k \frac{q_1 \cdot q_2}{\epsilon r^2}$
>> где:
>>> k - электрическая постоянная
>>> r - расстояние между центрами тел

>[!example] Электрическое поле
> Взаимодействие заряженных тел происходит в том числе по средству **электрического** поля
>
> Источником поля (тем, что обосновывает конкретные математические преобразования над свойствами помещённого в точку пространства тела) является заряд - неподвижный или движущийся (*note: магнитное поле создаётся только движущимися зарядами*)
>
>> [!abstract] Напряжённость электрического поля
>> характеристика, определяющая насколько сильно влияет электрическое поле на помещённый в него единичный положительный (пробный) заряд.
>> $\vec{E} = \frac{\vec{F}}{q}$
>
>> [!abstract] Поле точечного заряда
>> характеристика, определяющая насколько сильно влияет электрическое поле на помещённый в него единичный положительный (пробный) заряд.
>> $E = k \frac{q}{\epsilon r^2}$
>
>> [!abstract] Силовые линии поля
>> это воображаемые кривые, которые используются для наглядного графического представления векторов в поле (а именно: их направления (т.к. поле направленное) и величины/силы (отражается через плотность силовых линий))
>
> однородное поле: $\vec{E} = const$

>[!example] Потенциал
> Электрическое поле **потенциально** - то есть работа сил поля по перемещению заряда зависит только от начального и конечного положения, вне зависимости от траектории перемещения в поле.
> это значит, что заряд, помещённый в электрическое поле, обладает потенциальной энергией:
>
>> [!abstract] Потенциальная энергия помещённого заряда
>> Работа, которую необходимо применить, чтобы вывести заряд в область нулевой потенциальной энергии (то есть равно высвобождённой работе, которую совершит поле при противодействии перемещению заряда)
>> Для заряда в однородном электрическом поле с напряжённостью $\vec{E}$:
>> $W = qEd$
>> т.к. $d$- перемещение вдоль силовой линии и $F =qE$
>
>> [!abstract] **Потенциал**
>> скалярная величина, отражающая то, какую энергию имеет единичный положительный заряд (т.к. потенциальная энергия пропорциональна заряду и увеличивается вместе с тем, как увеличивается заряд)
>> $\varphi = \frac{W}{q}$
>> также следствием является:
>> $\varphi = \frac{kQ}{r}$, где $Q$ - источающий поле заряд
>
>> [!abstract] Разность потенциалов
>> Отражает то, насколько большая работа требуется для перемещения из точки с потенциалом 1 в точку с потенциалом 2 единичного точечного положительного заряда
>> $u = \varphi_1 - \varphi_2$
>>> [!warning] **Важно***
>>> $u > 0$ - строго. 
>>> в ином случае, мы говорим о разности потенциалов между точкой 2 и точкой 1, т.к. напряжение это переход из точки с большим потенциалом в меньший
>
>> [!abstract] Работа по перемещению зарядов
>> та работа, которая высвобождается при перемещении тела с зарядом $q$ между точками с разностью потенциалов $u$.
>> $A_\text{12} = qU$
>>
>> Так как потенциал отражает потенциальную энергию поля в точке, также верно:
>> $A = - \Delta W$
>
>> [!abstract] Связь напряжённости поля и напряжения
>> $u = Ed$
>> напомню, $d$ - расстояние вдоль силовой линии. 
>
>> [!abstract] Эквипотенциальные поверхности
>> между точками на равном расcтоянии от точечного источника (и не только, надеюсь понимаете) наблюдается ситуация, где перемещение между этими точками заряда ближе или дальше к нулевой потенциальной энергии заряд не сделает; по определению потенциальности поля, это значит, что в точках потенциальная энергия одинакова.
>> Поверхности (в проекции - прямые/кривые/окружность), обладающие равным потенциалом также называются ***эквипотенциальными***.

>[!example] Принцип суперпозиции электрического поля
> Как и в случае с силами, на помещённое тело может действовать поле, источаемое множеством зарядов. 
> Величина воздействия результирующего поля (их объединения), вычисляется как векторная сумма напряжённостей.
> 
> Так как потенциал отражает энергию, которая пропорциональная напряжённости поля, то для потенциала принцип сложения (суперпозиции) векторов тоже верен.
>
>> [!abstract] формула
>> $\vec{E} = \vec{E_1} + \vec{E_2} + ...$
>> $\varphi = \varphi_1 + \varphi_2 + ...$

>[!example] Проводник в электрическом поле
> На заряды внутри помещённого в поле проводника действует внешнее поле, отчего положительные (в твёрдых телах это "[дырка](https://en.wikipedia.org/wiki/Electron_hole)") и отрицательные заряды перераспределяются.
> На положительный заряд действует сила Кулона **ПО** направлению силовых линий, а потому они перемещаются вдоль силовых линий (в конденсаторе это к -), а отрицательное заряженные части - наоборот, против силовых линий.
>
> Перераспределённые заряды в результате формируют обратные внешнему полю полярности, отчего внутри проводника создаётся противоположное по направлению электрическое поле. Потому по принципу суперпозиции внутри проводника поле отсутствует (внешнее гасится внутренним)
> Также, из-за этого потенциал внутри проводника одинаков
>
>> [!note] тестовая часть
>> если поляризованный проводник разделить на две части, то каждая часть будет иметь свой собственный заряд. В сумме они будут равны первоначальному.
>> если же после этого поле убрать, то заряды, которые были в одной из частей, останутся в ней же.

>[!example] Диэлектрик в электрическом поле
> В диэлектрике отсутствуют **свободные** носители заряда, а потому для каждой из его не электронейтральных молекул (например, вода) будет наблюдаться то же, что и с отдельным проводником: они будут поворачиваться + к - (и наоборот), тем самым выстраиваясь по направлению силовых линий. Из-за этого на поверхности диэлектрика с разных сторон будет наблюдаться небольшой по величине заряды.
> Иными словами, сами-то заряды не переходят в диэлектрике, однако носители этих зарядов выстраиваются так, что с концов появляются некомпенсированные заряды

>[!example] Конденсатор
> В общем случае - две разноимённо заряженные поверхности на некотором удалении друг от друга.
> Плоский конденсатор - две плоскости на равном расстоянии.
>
> Из-за такой конструкции, внутри конденсатора образуется **однородное** электрическое поле.
>
>> [!abstract] Ёмкость конденсатора
>> характеризует способность накапливать на обкладках заряд; в отличие от цепей заряд между обкладками не течёт, но электростатическое поле между обкладками изменяется.
>> $Cu = q$
>> *note: не зависит от величины заряда!*
>
>> [!abstract] Ёмкость плоского конденсатора
>> $C = \frac{S \epsilon \epsilon_0}{d}$
>> где:
>>> $S$ - площадь обкладки
>>> $d$ - расстояние между обкладками
>>> $\epsilon$ - диэлектрическая проницаемость среды в конденсаторе
>>> $\epsilon_0$ - электрическая постоянная
>
>> [!abstract] Энергия конденсатора
>> Энергия, запасаемая конденсатором из-за накопления заряда на обкладках и создания электростатического поля (которое имеет энергию)
>> $W_c = \frac{qU}{2} = \frac{Cu^2}{2} =\frac{q^2}{2C}$

>[!example] Параллельно-последовательное соединение конденсаторов 
>
>> [!abstract] Параллельное
>> $q = q_1 + q_2 + ...$
>> $C = C_1 + C_2 + ...$
>
>> [!abstract] Последовательное
>> $q = q_1 = q_2 = ...$
>> $\frac{1}{C} = \frac{1}{C_1} + \frac{1}{C_2} + ...$
