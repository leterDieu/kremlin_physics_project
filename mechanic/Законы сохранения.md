>[!example]+ некоторые величины
>
>> [!abstract] Импульс
>> $\vec{p} = m\vec{V}$
>
>> [!abstract] Работа по перемещению
>> $A = |\vec{F}| |\vec{S}| \cos(\alpha)$
>> где $\alpha$ - угол между перемещением и силой
>
>> [!abstract] Мощность
>> $P = \frac{A}{t}$
>
>> [!abstract] Кинетическая энергия тела
>> $E_\text{к} = \frac{mV^2}{2}$
>
>> [!abstract] Потенциальная энергия тела в поле тяжести
>> $E_\text{п} = mgh$

>[!example]+ Закон об изменении кинетической энергии
>
>> [!abstract] формула
>> $A = E_\text{к2} - E_\text{к1}$

>[!example]+ Законы изменения и сохранения механической энергии (ЗСЭ)
>
>> [!abstract] Закон изменения механической энергии
>> $\Delta E_\text{мех} = A_\text{непотенц. сил}$
>
>> [!abstract] Закон сохранения механической энергии
>> условие: $A_\text{непотенц. сил} = 0$
>> $E_\text{мех} = E_\text{к} + E_\text{п} = const$

>[!example]+ Законы изменения и сохранения импульса (ЗСИ)
>
>> [!abstract] Закон изменения импульса
>> $\Delta \vec{p} = \vec{R} \cdot \Delta t$
>> где $\vec{R}$ - равнодействующая внешних сил, действующих на систему
>
>> [!abstract] Закон сохранения импульса
>> $\Delta \vec{p} = \vec{0}$
>> Выполняется тогда и только тогда, когда система замкнута: внешние силы не влияют на энергию системы (либо влияют незначительно); потерь энергии нету.
>> условия:
>>> $\Delta t \to 0$ - время изменения импульса крайне мало
>>> $\vec{R}$ пренебрежимо мала (0) в сравнении с внутренними силами реакции (например, взрывными или силами соударения)
>> $E_\text{мех} = E_\text{к} + E_\text{п} = const$
>>
>>> [!note] важно:
>>> система может быть замкнута не полностью, а, например, по оси Ox ($\vec{R_x} = \vec{0}; \vec{R} \neq \vec{0}$): тогда $\Delta \vec{p_x} = 0$
