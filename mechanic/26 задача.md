>> [!abstract] step 1
>> **условие:** Почти в каждой задаче (99/100) на механику требуется второй закон ньютона или условия равновесия. потому пишется всегда
>> **текст:** будем решать задачу в системе отсчёта (с.о.) относительно Земли. Эта с.о. инцерциальна
>
>> [!abstract] step 2
>> **условие:** применение законов ньютона (силы должны действовать из одной точки)
>> **текст:** Тело движется поступательно, а потому для описания его движения воспользуемся моделью материальной точки.
>> 
>> **дополнение:** необходимо нарисовать рядом с основным чертёжом "вынесенный" через стрелку чертёж, где силы будут приложены к БУКВАЛЬНО точке.
>
>> [!abstract] step 3
>> **условие:** наличие нити и требование подтверждения **отношения** ускорения нескольких тел, связанных нитью (блок и груз, и проч.)
>> **текст:** нить нерастяжима, а значит (для понимания - отрезки нити l1 и l2) $l_1 + l_2 = const$ => $\Delta l_1 + \Delta l_2 = 0$ => $\frac{\Delta l_1}{\Delta t} + \frac{\Delta l_2}{\Delta t} = 0$ => $V_1 + V_2 = 0$ (notice: характер + или - зависит от того, удлинятся нить или укорачивается) => $\frac{\Delta V_1}{\Delta t} + \frac{\Delta V_2}{\Delta t} = 0$ => $a_1 = - a_2$ 
>> 
>>> [!warning] Внимание:
>>> Здесь $+V_1$ и т.д. - относительная скорость концов нити. То есть $\frac{\Delta l_1}{\Delta t}$ может быть равна более сложной зависимости, нежели скорости одного из своих концов (V1 тут). 
>>> Зависимость a1 и a2 может быть более сложная. В общем случае, для подвижных блоков a_подвес = 2\*a_блок - однако это требуется получить обоснованием выше.
>>
>>>[!note] если блок неподвижен относительно груза
>>> *В этом случае достаточно сказать:* нить соединяет непосредственно блок и груз; нить нерастяжима => $a_\text{блока} = a_\text{груза}$
>

>> **текст:** нить невесома, а блоки идеальны => $T_1 = T_2$ - силы натяжения нити обоих концов

>>  $\vec{a_н} + \vec{a_\tau} = \vec{a}$
>>  $\vec{a} = \vec{a_н}$
>>  и тут же можно после указать: $a_н = \frac{\mathcal{V} ^2}{R}$
>
>> [!abstract] step 6
>> **условие:** требуются формулы РУД
>> **текст:** $a = const$ во время движения. Следовательно, его движение является равноускоренным, а потому применимы формулы равноускоренного движения
>
>> [!abstract] step 7
>> **условие:** тела движутся вместе и имеют негладкую поверхность
>> **текст:** Тела движутся вместе => по третьему закону Ньютона $F_\text{тр тела1} = F_\text{тр тела2}$
>

>>  тут много пунктов

>> **текст:** (СОКРОЩЁННО АТТ)
>> 1. Форма и размеры тела неизменны, т.е. расстояние между любыми двумя точками не меняется, а потому для описания его движения воспользуемся моделью абсолютно твёрдого тела.
>> 2. (если есть сила тяжести/архимеда) Стержень (или жидкость) однороден, а потому $F_\text{тяж}$ приложена к геометрическому центру (поставить точку) - т. Е
