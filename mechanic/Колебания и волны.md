>[!example]+ Определения
> $T = \frac{t}{N}$ - время, требуемое на одно событие
> $\nu = \frac{N}{t}$ - количество раз события в секунду
> $\omega = 2\pi\nu$ - циклическая частота
> $A$ - амплитуда; максимальное отклонение от позиции равновесия
> $\varphi$ - фаза колебания. нынешнее состояние отклонение от равновесия

>[!example]+ Связь периода и частоты
> $T = \frac{1}{\nu}$

>[!example]+ Кинематическое описание колебаний
>
>> [!abstract] Координата 
>> $x(t) = A\sin{(\omega t + \varphi)}$
>
>> [!abstract] Скорость 
>> $V(t) = x'(t)$
>
>> [!abstract] Ускорение 
>> $a(t) = x''(t) = V'(t)$
>
>> [!abstract] Амплитуды скорости и ускорения 
>> $V_\text{max} = \omega A$
>> $a_\text{max} = \omega ^2A$

>[!example]+ Энергетическое описание пружинного маятника
>
>> [!abstract] формула 
>> $\frac{mV^2}{2} + \frac{kx^2}{2} = \frac{mV_\text{max}^2}{2} = \frac{kA^2}{2}$

>[!example]+ Период свободных колебаний

>> $T = 2\pi\sqrt{\frac{l}{g}}$
>
>> [!abstract] Пружинного маятника 
>> $T = 2\pi\sqrt{\frac{m}{k}}$
