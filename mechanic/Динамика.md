>[!example]+ Законы механики Ньютона
>
>> [!abstract] Первый закон Ньютона
>> Существует Инерциальная Система Отсчёта (ИСО), в которой пока на тело не действуют другие тела или нету компенсации этого действия тело будет сохранять состояние покоя или прямолинейного движения
>
>> [!abstract] Второй закон Ньютона
>> $\vec{F_\text{р}} = ma$, где $\vec{F_\text{р}}$ - равнодействующая всех сил
>> *note: работает только в ИСО*
>
>> [!abstract] Третий закон Ньютона
>> $\vec{F_\text{12}} = -\vec{F_\text{21}}$
>> сила действия равна силе противодействия
>>> [!note] важно
>>> в задачах по механике часто ошибаются: "$N = mg$ по третьему з-ну Ньютона". Важно понимать, что это так по **второму** з-ну на определённую ось.

>[!example]+ Закон Гука
>> [!abstract] Сила упругости
>> $F_x = -k\Delta x$
>> *знак "-", вроде как, не обязателен. Силу упругости можно вводить не только спроецировано на ось*

>[!example]+ Закон Амонтона-Кулона
>> [!abstract] Сила трения скольжения
>> $F_\text{тр} = \mu N$
>
>> [!abstract] Сила трения покоя
>> $F_\text{тр} \le \mu N$
>> *точно силу трения покоя можно найти только через второй закон Ньютона.*

>[!example]+ Закон всемирного тяготения
>> [!abstract] Сила притяжения двух тел
>> $F = G\frac{m_1 m_2}{r^2}$
>> где: 
>>> G - гравитационная постоянная
>>> r - расстояние между центрами
>>> m - массы тел
>
> также полезны выводы:
>
>> [!abstract] Первая космическая
>> скорость, которую необходимо развить, чтобы удерживать стабильную круговую орбиту вокруг космического тела (т.е. не сваливаться на него)
>> $a_\text{цс} = \frac{\mathcal{V}^2}{R}$
>> $F = ma$
>> $F = G\frac{m_1 m_2}{r^2}$
>> следовательно (опуская подсчёты):
>> $\mathcal{V} = \sqrt{\frac{Gm_2}{R}}$
